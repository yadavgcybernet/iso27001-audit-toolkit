## How to use this checklist

For each control, mark the implementation status using the key below.
Document evidence references and any findings in the columns provided.

### Status key

| Symbol | Status | Meaning |
|---|---|---|
| ✅ | Implemented | Control is fully in place with documented evidence |
| 🟡 | Partial | Some evidence exists but gaps remain |
| ❌ | Not Implemented | No evidence of implementation |
| N/A | Not Applicable | Excluded in SoA with documented justification |
